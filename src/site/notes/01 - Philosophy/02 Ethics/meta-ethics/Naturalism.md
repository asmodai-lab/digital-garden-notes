Naturalism
{ #7Tdw9hhT}


Ethical naturalism says that moral properties are natural properties.
{ #IStP3h7f}


Naturalism argues that moral judgements are beliefs that are intended to be true or false (cognitivism) and that moral properties exist (realism) and are natural properties.
{ #DvhwHEjG}


So, the statement "murder is wrong" expresses a cognitive belief that murder is wrong, where 'wrong' refers to a natural property.
{ #lvBsad2D}


A natural property is something like grass having the property of being green, or a knife having the property of being metal.
{ #EQIGTsy1}


Think Aristotle's four causes
{ #xErFb7fs}


Mill's 'proof' of Utilitarianism
{ #sBhCwIlN}


J.S. Mill argues that happiness is not only good, but it is the only good.
{ #KvMAjeSr}


Other things we desire - such as truth and freedom - are part of what happiness is to us.
{ #tzpXtFTO}


Mill's metaethical argument for utilitarianism:
{ #maVNaFAM}


- The only proof that something is desirable is that people desire it
- no proof can be given for why general happiness is desirable other than the fact that each person desires happiness
- this desirability is "all the proof the case admits of" that happiness is a good thing
- all our other values (truth, freedom, dignity etc) constitute what make us happy
-- in other words, the reason we value these things is because they make is happy
- therefore, not only is happiness good, it is the only good.
{ #va2hs8Ka}


Virtue ethics as naturalism
{ #9F5CVWHB}


Although utilitarianism is the obvious form of moral naturalism, some see virtue ethics as a form of moral naturalism, too.
{ #H9oPjVi2}


For example, Aristotle's ergon/function
{ #1lnZ9frc}


Aristotle sees everything as having a function - or ergon. Things are good if they fulfil their function, and bad if they don't.
{ #Lh7SZVJY}


A knife's ergon is to cut things
{ #sWdj4L7k}


A human's function is to use reason. This does not mean we achieve eudamonia by just thinking, but that we use reason in accordance with our virtues (good character traits)
{ #jooGYxqM}


Just as sharpness helps a knife fulfil it's ergon to cut, virtues help humans to fulfil their ergon, which is to choose actions according to reason.
{ #ik0HqkY7}


This can be interpreted as a discussion of natural facts about human beings.
{ #s3geqD3B}


We might argue that it is a natural fact that the function of human beings is to use reason - the same way it is a natural fact that the function of a knife is to cut things.
{ #nc5WFUoD}


There is nothing non-natural about claiming that the function of a knife is to cut things, and likewise nothing non-natural about claiming that the function of human beings is to use reason.
{ #udHq0PZv}


So, on this reading, 'good' reduces to a set of natural facts about function and performing that function well. 
{ #c1td9tnl}


For example, being courageous is good because being courageous helps humans act correctly according to reason.
{ #ygJhUrey}

The Naturalistic fallacy
{ #JLTpQ9tg}


This is the fallacy of equating goodness with some natural property (such as pleasure/pain)
{ #Gu70Jsyn}


For example, Moore says it is a fallacy to conclude that drinking beer is good from the fact that drinking beer is pleasurable.
{ #tnopsJ3f}


These are two different categories of properties - even if they are correlated, it can't follow that they are the same thing.
{ #7WZn1Lcs}


From this, Moore would argue that Mill's proof is invalid: To conclude that happiness (a natural property) is 'good' (a moral property) commits the naturalistic fallacy.

The is/ought problem
{ #d2cNHdiO}


'is' statements
{ #AbxES7CC}


'ought' statements
{ #yXnulEJG}


Factual claims about what is the case
{ #msBseo16}


Value judgements about what should be the case.
{ #RotyQg8b}


E.g. "Smith murdered Jones."
{ #Y0gb6XFs}


E.g. "Smith shouldn't have murdered Jones"
{ #X79Tc2ZP}


Hume argues that there is a gap between the two kinds of claim: you cannot logically derive ought statements, like 'you ought not torture' from statements about what is, like 'that is an act of torture.'
{ #J7Tln0a0}


We can argue that this is evidence for non-cognitivism: the reason we cannot derive 'ought' statements from 'is' statements because the former is non-cognitive and the latter is cognitive.
{ #PBK5vjlz}


Hume would say 'is' statements like "Smith murdered Jones" are capable of being true or false, whereas 'ought' statements like "Smith shouldn't have murdered Jones" are expressions of emotion that are not capable of being true or false.

A.J. Ayer: Language, truth, and logic
{ #DrhRtKXq}


Ayer's verification principle is basically just Hume's fork with different words.
{ #t4Lm8ikD}


Ayer argues that moral judgements fail the verification principle. "Murder is wrong" is clearly not an analytic truth, and Ayer argues that it isn't empirically verifiable either - both on the naturalist and non-naturalist interpretations:
{ #Q4MDC2dN}


Naturalism
{ #dzZjnstM}


non-naturalism
{ #9IIsrV6U}


Naturalism would argue that we can prove that murder causes pain/anger etc. However, Ayer would argue that this isn't the same as proving murder is wrong. We can empirically verify that murder causes pain, but we cannot empirically verify that murder has the property of 'wrongness'
{ #YD2dzxwI}


Ayer also argues that there is no way to empirically verify the presence of non-natural properties - i.e. supernatural properties.
Even if "murder is wrong" did possess the non-natural property of wrongness, how could we ever prove this? The existence of non-natural properties cannot be empirically proven.
{ #MF6gikWV}


Therefore, Ayer concludes that moral judgements simply express feelings of approval or disapproval, and seek to evoke the same feeling in others.
{ #KmZOLIKQ}
